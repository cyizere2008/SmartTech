
# SmartTech HRMS

## Features
- JWT Authentication
- Employee Management
- Department Management
- Attendance Tracking
- Leave Management
- Payroll System
- Performance Reports
- PDF Reports

## Technologies
Frontend:
- React.js
- JSX
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Roles
- HR Manager
- Employee
- Administrator
